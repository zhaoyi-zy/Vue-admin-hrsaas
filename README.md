# vue-admin-hrsaas

## 介绍

vue-admin-hrsaas是一个企业中后台的人资管理系统。

## 首页

安装使用
clone代码到本地

```
git clone https://github.com/zhaoyi-zy/Vue-admin-hrsaas.git
```

## 运行

### 进入文件

```
cd vue-admin-hrsaas
```

### 安装依赖

```
yarn install
```

### 启动项目

```
yarn run dev
```

### 浏览项目

> 浏览器中输入 http://localhost:8888 即可以访问
